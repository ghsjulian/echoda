### About This Project

**Echoda is a complete multi-component MERN system consisting of:  
1) React Client  
2) React Admin Dashboard  
3) Node.js API Server  
4) Nginx Reverse Proxy & VPS Deployment Configuration**

**This project helped me strengthen full production-level skills covering scalable backend structure, JWT authentication, multi-component routing, Nginx load balancing, PM2 process management, and secure server deployment practices.**

**It includes advanced functionalities such as token-based APIs, user access management, product management, admin roles, secure data flow between client and server, and highly optimized production builds.**

**For more details, view the sections below and see the project live demo.**

### How To Clone :

```bash
git clone https://github.com/ghsjulian/echoda.git
cd echoda

# server setup
cd echoda-api
npm install
npm run dev  # or npm start

# client setup
cd ../echoda-client
npm install
npm run dev

# admin setup
cd ../echoda-admin
npm install
npm run dev

```
